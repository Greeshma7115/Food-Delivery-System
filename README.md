# Food Delivery System (Java Console Application)

A console-based Food Delivery System developed using Core Java and Object-Oriented Programming (OOP) principles.  
This project simulates real-world food ordering functionality including restaurant selection, cart management, quantity updates, and payment processing.

---

## Features

-  Multiple Restaurant Support
-  Dynamic Menu Handling (No hardcoded options)
-  Cart Management using HashMap (Grouped quantity display)
-  Update Item Quantity (Set new quantity or remove)
-  Payment System using Interface & Polymorphism
-  Persistent Order ID (Auto-increment even after restart)
-  Order Storage using File Handling
-  Input Validation (Name & 10-digit mobile number)
-  Cancel Order Option

---

##  Technologies Used

- Java
- OOP (Encapsulation, Abstraction, Polymorphism)
- Collections (HashMap, ArrayList)
- File Handling
- Git & GitHub
